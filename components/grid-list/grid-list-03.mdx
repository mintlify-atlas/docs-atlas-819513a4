A responsive grid list component that displays action cards or feature sections with custom icons and color schemes. Ideal for help centers, feature showcases, or navigation hubs.

## Installation

<CodeGroup>
```bash npm
npx shadcn@latest add https://blocks.so/r/grid-list-03.json
```

```bash pnpm
pnpm dlx shadcn@latest add https://blocks.so/r/grid-list-03.json
```

```bash yarn
npx shadcn@latest add https://blocks.so/r/grid-list-03.json
```

```bash bun
bunx shadcn@latest add https://blocks.so/r/grid-list-03.json
```
</CodeGroup>

## Dependencies

This component requires the following dependencies:

- `card` - shadcn/ui card component
- `lucide-react` - Icon library

## Usage

```tsx
import GridList03 from "@/components/grid-list-03";

export default function Page() {
  return <GridList03 />;
}
```

## Features

- Responsive grid layout (1 column on mobile, 2 on tablet, 3 on desktop)
- Custom color-coded icons with ring borders
- Dark mode support with tailored color schemes
- Arrow indicator on hover for visual feedback
- Entire card is clickable using stretched link pattern
- Focus ring for accessibility
- Gap-based grid separator effect

## Customization

You can customize this component by:

- Modifying the `actions` array with your own content
- Changing icon colors using Tailwind color utilities
- Adjusting the grid layout with different column configurations
- Replacing Lucide icons with your preferred icon library
- Modifying the card spacing and border radius
- Adding animations or transitions for enhanced interactivity
- Implementing actual navigation handlers
