<Note>
This component features a Dialog-based command menu with extensive keyboard shortcuts and multiple action categories.
</Note>

## Installation

Install the component using the CLI:

```bash
npx shadcn@latest add command-menu-02
```

## Features

- **Keyboard Shortcuts**: Displays shortcuts using Kbd and KbdGroup components
- **Multiple Categories**: Documents, Signing, Templates, General, Navigation, Quick Actions, and Help
- **Custom Dialog**: Uses Dialog component with custom styling
- **Controlled Input**: Tracks input value state
- **ESC in Header**: Shows ESC shortcut in the header for quick access
- **Extensive Actions**: Over 30 different commands organized by category

## Component Structure

The component includes eight main groups:

### Account Actions
Account Settings, Switch Workspace, and Log Out with their respective keyboard shortcuts.

### Documents
Search, create, and upload documents with keyboard shortcuts like `⌘F`, `⌘N`, and `⌘U`.

### Signing
Request signatures, sign documents, and bulk send for signatures.

### Templates
Search and create templates.

### General
Theme changes and URL copying with shortcuts.

### Navigation
Quick navigation to Inbox, Action Required, Waiting for Others, Completed, Drafts, Templates, Archive, Trash, and Settings.

### Quick Actions
Copy signing links and download documents.

### Help
Search help center, send feedback, and contact support.
