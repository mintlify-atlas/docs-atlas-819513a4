Stats cards featuring circular progress indicators. Visualizes completion percentages with attractive radial progress bars.

## Installation

Install the component from the Blocks registry:

```bash
npx shadcn@latest add @blocks/stats-07
```

## Dependencies

This component requires the following shadcn/ui components:

- `card`
- `progress` (circular variant)

## Usage

```tsx
import Stats07 from "@/components/stats-07";

export default function Page() {
  return <Stats07 />;
}
```

## Features

- **Circular progress** - Visual percentage representation
- **Radial indicators** - Eye-catching progress visualization
- **Percentage values** - Clear numeric display
- **Multiple metrics** - Grid layout for comparison
- **Modern design** - Contemporary UI patterns

## Use Cases

- Goal tracking
- Completion metrics
- Resource utilization
- Progress monitoring
- Achievement tracking
