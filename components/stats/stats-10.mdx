Stats cards featuring area charts for trend visualization. Combines numeric metrics with visual trend data.

## Installation

Install the component from the Blocks registry:

```bash
npx shadcn@latest add @blocks/stats-10
```

## Dependencies

This component requires the following shadcn/ui components:

- `card`
- `chart` (area variant)

## Usage

```tsx
import Stats10 from "@/components/stats-10";

export default function Page() {
  return <Stats10 />;
}
```

## Features

- **Area charts** - Visual trend representation
- **Trend analysis** - Historical data visualization
- **Multiple metrics** - Compare trends side-by-side
- **Responsive charts** - Mobile-friendly visualizations
- **Data-driven** - Chart integration

## Use Cases

- Time-series data
- Trend monitoring
- Historical analytics
- Performance tracking over time
- Revenue visualization
